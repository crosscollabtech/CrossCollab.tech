# CrossCollab

**Code with purpose. Build for the Kingdom.**

CrossCollab is an open, community-driven platform that connects **ministries, NGOs, churches, and missionaries** with **Christian technologists** to build open-source software that serves real needs around the world.

All projects are public, collaborative, and GitHub-first.
